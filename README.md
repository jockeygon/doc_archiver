# 文档归档工具

根据 PDF/Word 等文档的文件名，**自动识别公司或大学名称**，创建文件夹并归档。

## 功能特性

- 🏢 **智能识别公司名**：`腾讯科技有限公司`、`阿里巴巴（中国）有限公司`（保留括号）
- 🎓 **智能识别大学名**：`北京大学`、`清华大学`、`复旦大学附属中学`
- 📎 **自动去除描述性后缀**：合同、协议、证明、推荐信、申请表、批复等
- 🔄 **自动去除描述性括号**：（以接收）、（暂定）、（草稿）等，**公司名中的括号会被保留**
- 📁 **文件夹自动重名处理**：如果文件夹已存在，自动追加时间戳
- ⚙️ **多种运行模式**：交互模式、自动模式、预览模式

## 使用方法

### Windows exe（推荐）

下载 Releases 页面中的 `Windows/文档归档工具.exe`，双击运行即可。

### 命令行运行

```bash
python3 doc_archiver.py <目录路径>        # 交互模式
python3 doc_archiver.py <目录路径> --auto  # 自动归档
python3 doc_archiver.py <目录路径> --dry   # 预览（不执行）
```

### 支持的文件类型

`.pdf` `.docx` `.doc` `.xlsx` `.xls` `.pptx` `.ppt` `.txt` `.md` `.csv` `.zip` `.rar`
